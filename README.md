# Stage_HIRACLES
Repo' contenant le code utilisé pour le projet HIRACLES

Entrainer le model : 
python train.py --dataset facades --cuda

Tester le model : 
python test.py --dataset facades --cuda

Installatio nécessaire (Dans l'ordre) :
- Python
- pip
- torch (pip)
- torchvision (pip)
- pillow (pip)
- torch.utils (pip)
- numpy (pip)
